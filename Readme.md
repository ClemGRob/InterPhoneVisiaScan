Sur la raspberry :
- git clone https://github.com/ClemGRob/InterPhoneVisiaScan.git
- cd InterPhoneVisiaScan

- Double clic Run_VisiaScan.sh
Message Execute File : "What do you want to do with it ?"
- Faire "Execute"
